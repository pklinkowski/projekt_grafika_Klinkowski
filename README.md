# Pong 3D, Paweł Klinkowski

## Funkcjonalności
- Renderowanie w pełnym 3D (OpenGL + GLU)
- Oświetlenie i materiały (GL_LIGHT0, specular, diffuse)
- Teksturowana podłoga i ściany
- Kolizje piłki z paletkami i granicami areny
- Efekt spinu zależny od miejsca odbicia
- Losowa pierwsza trajektoria piłki przy każdym punkcie
- Sterowanie dwoma paletkami
- Tryb kamery z góry
- Restart po punktach z krótką pauzą
- Gra do 5 punktów

## Jak uruchomić
- Pobierz i rozpakuj Release.zip
- Uruchom Klinkowski_Pawel_projekt.exe
  
## Sterowanie
- Ruch P1: ← / →
- Ruch P2: A / D
- Przełacz kamerę: C
- Wyjście: Esc
  
## Biblioteki
- SFML - obsługa okna, kontekst OpenGL, wczytywanie tekstur
- OpenGL - renderowanie 3D
- GLU - generowanie sfery (quadric)
